# サムネイルからの写真選択

```zsh
mkdir -v tmp
feh -t -E 384 -y 384 -W 1156 --action1 "mv -v  %f tmp/" --action "feh -F %f"
```

|オプション | 説明                                        |
|:---       | :---                                        |
| -t        | サムネイルモードへ                          |
| -E, -y    | サムネイルの最大サイズ                      |
| -W        | サムネイルモードウィンドウの横サイズ        |
| --actoin1 | 1 を押したときの動作。gomi とかでも良いかも |
| --action  | 0 を押したときの動作。とりあえず表示        |

キーボードでの操作は

|キー                | 説明                   |
|:---                | :---                   |
| ctrl-up, ctrl-down | スクロール             |
| n, p               | 次または前の画像を選択 |
| 0                  | 選択画像表示           |
| 1                  | 選択画像 tmp/ に移動   |
| x                  | 画像表示から戻る       |
| q                  | 終る                   |
